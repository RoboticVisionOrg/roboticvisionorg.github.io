Robotic Vision

Installing Software

Added the Repositories

Import the RV GPG Key using the following command 

sudo -E apt-key adv --keyserver hkp://keyserver.ubuntu.com --recv-key 5B76C9B0

Add the Robotic Vision repository to the apt sources list directory

sudo sh -c 'echo "deb [arch=$(dpkg --print-architecture)] http://roboticvision.org/packages $(lsb_release -sc) main" > /etc/apt/sources.list.d/acrv-latest.list'

Update your packages list

sudo apt update

--- QUT Internal Repository ---

For internal testing purposes, the following repository contains the most recent builds of the RV packages.

sudo sh -c 'echo "deb [arch=$(dpkg --print-architecture)] http://robotics.qut.edu.au/packages $(lsb_release -sc) main" > /etc/apt/sources.list.d/acrv-latest.list'

Note: This uses the same key as the public repositories.
